# hugo-coder-nord

[hugo-coder](https://github.com/luizdepra/hugo-coder) but with the [nord theme](https://www.nordtheme.com)

## Quick Start

Add the repository into your Hugo Project repository as a submodule, `git submodule add https://github.com/BrandonLynch2402/hugo-coder-nord themes/hugo-coder-nord`.
